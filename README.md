# weather-web-app
A responsive Weather App built using HTML, CSS, and JavaScript that provides real-time weather information such as temperature, humidity, and wind speed for any city using a weather API.

🚀 Live Demo:
https://github.com/sahu-jagrati/weather-web-app

✨ Features:
🌍 Search weather by city name
🌡️ Real-time temperature display
💧 Humidity and wind speed details
☁️ Weather condition (Cloudy, Sunny, Rainy, etc.)
⚡ Fast and responsive UI
📱 Works on mobile and desktop

🛠️ Tech Stack:
HTML – Structure
CSS – Styling
JavaScript – Functionality
Weather API – Fetch real-time data
